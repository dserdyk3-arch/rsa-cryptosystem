## Описание

Программная реализация криптосистемы RSA на Python с поддержкой больших чисел.

### Возможности

- Генерация ключей (тест Миллера–Рабина)
- Сохранение и загрузка ключей из файлов
- Шифрование и расшифрование файлов
- Атака на малую экспоненту e=3 (не переборная)
- Ручная демонстрация из отчёта

---

## Установка и запуск

1. Убедитесь, что установлен Python 3.8 или выше
2. Скачайте файл `rsa_crypto.py`
3. Запустите в терминале/командной строке:
python rsa_crypto.py

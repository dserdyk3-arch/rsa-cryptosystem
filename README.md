## Описание

Программная реализация криптосистемы RSA на Python с поддержкой больших чисел.

### Возможности

- Генерация ключей (тест Миллера–Рабина)
- Сохранение и загрузка ключей из файлов
- Шифрование и расшифрование файлов
- Атака на малую экспоненту e=3 (не переборная)
- Ручная демонстрация из отчёта

## Установка и запуск

1. Убедитесь, что установлен Python 3.8 или выше
2. Скачайте файл `rsa_crypto.py`
3. Запустите в терминале/командной строке:
python rsa_crypto.py

Атака на малую экспоненту e=3 (не переборная)
Если e=3 и сообщение m достаточно маленькое (m³ < n), то шифрование выполняется без модуля:

c = m³ mod n = m³
Злоумышленник, перехвативший шифртекст c, извлекает кубический корень и восстанавливает сообщение:

text
m = ∛c
Пример:

m = 42

n = 100000 (большой модуль)

e = 3

c = 42³ = 74088 (меньше n, поэтому модуль не сработал)

Атака: ∛74088 = 42 → сообщение восстановлено

Вывод: данная атака не является наивным перебором, она основана на математическом свойстве. Поэтому на практике используют e=65537 и схемы дополнения (padding).
